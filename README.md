# Matthew Ramer's Professional Website

Welcome to my professional website! This website serves as a portfolio showcasing my skills, projects, and experiences.

## About

This website is built using React and hosted on GitHub Pages. It also showcases functionality of Framer Motion.

## Usage

### `npm start`

Runs the website locally.\
Open <a href="http://localhost:3000" target="_blank">http://localhost:3000</a> to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.


### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run deploy`

Deploys the current build to github pages branch. This will update <a href="https://matthewramer.github.io/" target="_blank">my website</a> with the next push.
